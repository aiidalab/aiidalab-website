FAQ

How can I get access to the Materials Cloud deployments?
These deployments are restricted to members of organizations affiliated with the Materials Cloud partners. If you are one of these members, please contact aiidalab@materialscloud.org and request access.
I am not eligible for any of these deployments, how can I use AiiDAlab?
The easiest would be to simply run it yourself! Please see here for instructions on how to run AiiDAlab yourself.
Questions?
Please message us at aiidalab@materialscloud.org for any other questions.
